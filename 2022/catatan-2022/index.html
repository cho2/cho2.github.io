Catatan Akhir Tahun 2022

Ternyata sepanjang tahun 2022 ini saya tidak menulis satu pos apapaun di blog ini karena kondisi sedang tidak baik, maka di akhir tahun ini saya coba rangkum, kalau ada bagian yang terlewat, mungkin saya terlupa, mohon maaf. Berikut rangkuman kegiatan FOSS yang saya ikuti di 2022.

Maret #

GNOME 42 #

Rilis GNOME 42, kontributor tetap di l10n.

Juni #

openSUSE Conference #

openSUSE Conference 2022, meramaikan daring saja.

Juli #

GUADEC #

GUADEC 2022 berlangsung hybrid, daring dan tatap muka di Guadalajara, Mexico. Hadir daring di meet.gnome.org, hanya beberapa sesi saja karena masalah zona waktu (mulai jam 21.00, selesai di 03.00 UTC+7).

openSUSE-ID #

15 tahun openSUSE-ID.

Agustus #

Nest With Fedora #

Nest with Fedora 2022, nongkrong-nongkrong aja di hopin.

LibreOffice Translation & QA #

LibreOffice Translation & QA bersama the one and only Pak Andika Triwidada selaku penerjemah FOSS seumur hidup. Nongkrong daring aja. Versi tatap mukanya ada di Bogor dan Gresik. Foto-foto lengkapnya ada disini.

September #

openSUSE.Asia Summit #

openSUSE.Asia Summit 2022, kali ini meramaikan saja via daring.

LibreOffice Conference #

LibreOffice Conference, hybrid, tatap mukanya di Milan, Italia sono, nongkrong daring aja.

GNOME 43 #

Rilis GNOME 43, (masih) kontributor tetap di l10n.

Oktober #

openSUSE-ID #

openSUSE-ID turun gunung juga akhirnya. Meetup tatap muka pertama. Belum bisa ikutan hadir, jadi bantu-bantu saja di belakang layar. Ringkasannya ada disini. Acara ini sebenarnya juga acara regional untuk openSUSE.Asia Summit 2022.

Akademy #

Akademy 2022, hybrid, tatap mukanya di Barcelona, Spanyol, nongkrong daring aja lagi.

November #

Fedora 37 Release Party #

Fedora Linux 37 Release Party, nongkrong-nongkrong aja di hopin (lagi).

ILC #

ILC (Indonesia Linux Conference) nongol lagi dengan segala keramaiannya, ulasan saya dapat dibaca disini.

Ubuntu Summit #

Ubuntu Summit 2022, ngrecokin aja perkara WSL, hehe.

UbuCon Asia #

UbuCon Asia 2022 tatap muka pertama nih di Seoul, Korea Selatan, hybrid juga sebenernya, meramaikan daring saja karena ada pembicara dari ID.

Desember #

GNOME.Asia #

GNOME Asia Summit 2022 di KL, MY, hybrid. Sepi, tidak meriah, kurang terorganisir untuk yang daring, padahal ada beberapa sesi yang menarik. meramaikan daring saja karena ada pembicara dari ID lagi.

openSUSE-ID #

openSUSE-ID bikin meetup akhir tahun. Masih belum bisa hadir, meramaikan via YouTube saja dari jauh dan bantu-bantu di belakang layar. Ringkasannya ada disini.

Btw, buat yang mau jadi pembicara atau memberikan dukungan tempat meetup, boleh kontak-kontak di humas@opensuse.id ya.

Lainnya #

Flatpak/Flathub #

Bantu-bantu di Flathub KDE, untuk KDE Edu dan Games. Juga app lain seperti Inkscape, dll.

Sampai jumpa di tahun 2023, mudah-mudahan kita semua sehat selalu!
